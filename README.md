## Гид по каждому скрипту

Запустить проект можно только через консоль. Способ пуска зависит только от консоли, которую вы используете(**ВСЕ ДАННЫЕ СПОСОБЫ БУДУТ РАБОТАТЬ ТОЛЬКО ЕСЛИ КОМПИЛЯТОР PYTHON НАХОДИТЬСЯ В PATH!!!**, подробнее по [ссылке](https://www.java.com/ru/download/help/path.html)). Все пути использованые в гиде условные:

### `artists.py`

Данный скрипт принимает на вход аргумент `author_id`(описание его работы можете найти, если после запроса поставите аргумент `--help`). Пример запроса к данному скрипту:

```sh
python D:/playlist/artists.py --author_id=20185
```

### `search.py`

Данный скрипт принимает на вход аргумент `request`(описание его работы можете найти, если после запроса поставите аргумент `--help`). Пример запроса к данному скрипту:

```sh
python D:/playlist/search.py --request='Kendrick Lamar'
```

### `songs.py`

Данный скрипт принимает на вход аргумент `song_id`(описание его работы можете найти, если после запроса поставите аргумент `--help`). Пример запроса к данному скрипту:

```sh
python D:/playlist/songs.py --song_id=1883888
```

### `tools.py`

Данный скрипт техническийи не может отдать вам никаких данных. Любое вмешательство неопытного человека в код может привести к неработоспособности некоторых скриптов
